# Flask Task App

## 概要
Flaskを使用して作成した、ログイン機能付きのタスク管理アプリです。  
ユーザー認証、セッション管理、SQLiteによるデータ保存など、  
Webアプリケーションの基本的な仕組みを学ぶ目的で個人開発しました。

---

## 機能一覧
- ユーザー登録
- ログイン / ログアウト
- タスクの追加
- タスクの削除
- ログイン状態のセッション管理
- パスワードのハッシュ化による安全な認証

---

## 使用技術
- Python 3
- Flask
- SQLite
- HTML / CSS
- Werkzeug（パスワードハッシュ）

---

## 画面イメージ
- ログイン画面
- タスク一覧画面  
（※簡易的なUIですが、機能理解を重視しています）

---

## 工夫した点
- パスワードを平文で保存せず、ハッシュ化して管理することで  
  セキュリティを意識した実装を行いました。
- ログインしていない状態ではタスク画面にアクセスできないよう  
  セッション管理を実装しました。
- CRUD処理（作成・取得・削除）を一通り実装し、  
  Webアプリの基本的な流れを理解しました。
